 # 🗺️ CoverMap — Analyse de couverture des délégués terrain

## 🚀 Démo en ligne

👉 **[Accéder à l’application CoverMap](https://covermap-io9xbtu7zlgsxklcbsofxa.streamlit.app/)**

---

## 🎯 Objectif

CoverMap aide les équipes terrain et les managers à :

- 📍 Visualiser les délégués sur une carte
- 📏 Analyser leur zone de couverture (rayon de 30 km)
- 👨‍⚕️ Identifier les médecins visités et non visités
- ⚠️ Détecter les zones non couvertes (zones blanches)
- 📈 Optimiser les tournées terrain

---

## 📊 Problématique métier

Les entreprises terrain manquent souvent de visibilité sur :

- Les zones réellement couvertes  
- Les médecins effectivement visités  
- Les zones sous-exploitées  
- L’efficacité des délégués  

👉 CoverMap apporte une solution basée sur la data et la visualisation.

---

## ⚙️ Fonctionnalités

### 🗺️ Cartographie interactive
- Affichage des délégués sur carte
- Zones de couverture (rayon de 30 km)
- Visualisation des médecins (réseau / hors réseau)

### 📋 Suivi des visites
- Enregistrement des visites terrain
- Ajout de commentaires
- Historique des visites

### 📊 Dashboard KPI
- Taux de couverture
- Nombre de délégués actifs
- Médecins couverts / non couverts

### 🔍 Filtres avancés
- Par zone
- Par spécialité
- Par statut
- Par délégué

### 📥 Export des données
- Export Excel
- Export carte HTML

## 🛠️ Stack technique

- *Python 3.10+*
- *Streamlit* — Interface web
- *Folium* — Cartographie
- *Pandas* — Manipulation des données
- *Google Sheets API* — Stockage des données
- *gspread* — Connexion API
- *openpyxl* — Export Excel
